# Slot Machine Sprite Spec

The reviewer uses generic internal slot ids instead of content names:

- `slot_1` 🐟
- `slot_2` 🍖
- `slot_3` 🔑
- `slot_4` 👑
- `slot_5` 🍀

Current sprite file mapping:

- `slot_1` 🐟 -> `slot_1.png`
- `slot_2` 🍖 -> `slot_2.png`
- `slot_3` 🔑 -> `slot_3.png`
- `slot_4` 👑 -> `slot_4.png`
- `slot_5` 🍀 -> `slot_5.png`

Rules:

- Format: PNG with transparent background
- Use a consistent visual scale across all five sprites
- Keep the subject centered
- Source sprite format: `256x256`
- Reel display size in the UI: `64x64`
